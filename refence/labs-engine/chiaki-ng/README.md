# [labs](https://streetpea.github.io/labs/)

An open source PlayStation remote play project serving as the next-generation of Labs with improvements and ongoing support now that the original Labs project is in maintenance mode only. [Click here to see the accompanying site for documentation, updates and more](https://streetpea.github.io/labs/).

Labs is a Free and Open Source Software Client for PlayStation 4 and PlayStation 5 Remote Play
for Linux, FreeBSD, OpenBSD, Android, macOS, Windows, Nintendo Switch and potentially even more platforms.
